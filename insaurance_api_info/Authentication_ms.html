Authenticate

Authenticate method will return the Token ID that needs to be echoed back in every subsequent request.
In case of session expiry or log outs, the authenticate method needs to be called again and the updated Token ID needs to be passed in every subsequent request.
Service URL (REST) to be used:
http://sharedapi.tektravels.com/SharedData.svc/rest/Authenticate
