InsuranceSearch Details

This method provides availability of Insurance option with the insurance cover details on the behalf of PlanType, PlanCoverage and dates selected. Client need to pass the PlanType, PlanCoverage, TravelStartDate and passenger details in request in response will get the available Insurance option with the insurance cover details.

Service URL (REST) to be used:

https://InsuranceBE.tektravels.com/InsuranceService.svc/rest/Search

Insurance Search Request
Level Parameter Data Type Description Mandatory/ Optional
1 EndUserIp String IP Address of the end user Mandatory
2 TokenId String Token ID Mandatory (Token Id to be echoed back from Authenticate response)
3 PlanCategory Enumeration Define the category of plan whether its Domestic plan. Possible values can be DomesticTravelPolicy = 1, OverseasTravelInsurance = 2 Mandatory
4 PlanCoverage Enumeration Define whether plan coverage area is US, NON us and India (Ex: US = 1 Non_US = 2, WorldWide = 3 India = 4 Asia = 5 Canada = 6 Australia = 7 SchenegenCountries = 8) Mandatory
5 PlanType Enumeration Define insurance PlanType is single trip or AnnualMultiTrip (Ex: SingleTrip = 1, AMT = 2) Mandatory
6 TravelStartDate Date Time Start Date of the requested Insurance Format: (dd/mm/yyyy) Mandatory
7 TravelEndDate Date Time Start Date of the requested Insurance Format: (dd/mm/yyyy) Mandatory
8 NoOfPax Integer Number of passenger for which insurance requested Mandatory
9 PaxAge[] Array Passenger Age Mandatory
Insurance Search Response
Level Parameter Data Type Description Mandatory/ Optional
1 Response Status Enumeration Define the status of response whether its successful, failed or Not set. Possible values can be (NotSet = 0, Successful = 1, Failed = 2, InCorrectUserName = 3, InCorrectPassword = 4, PasswordExpired = 5) Mandatory
2 Error Object Mandatory
2.1 ErrorCode Integer Code corresponding to the Error Mandatory
2.2 ErrorMessage String Contains error message (Will contain error description corresponding to ErrorCode.Error Message would be blankin case ErrorCode=0) Mandatory
3 TraceId String Unique identifier for identifying every unique search Required for audit purpose Mandatory (Trace Id to be echoed back in further requests)
4 Results[] Array Array of results (Can be blank in case no Insurance found for requested City) Mandatory
4.1 ResultIndex Integer Result index Mandatory
4.2 PlanType Enumeration Define insurance PlanType is Single trip or AnnualMultiTrip (Ex: SingleTrip = 1, AMT = 2) Mandatory
4.3 PlanType Enumeration Define insurance PlanType is Single trip or AnnualMultiTrip (Ex: SingleTrip = 1, AMT = 2) Mandatory
4.4 Plan Description String Describe the coverage amount and other details Mandatory
4.5 Plan Coverage String Define whether the plan coverage area is US, NON us and India (Ex: US = 1 Non_US = 2, WorldWide = 3 India = 4 Asia = 5 Canada = 6 Australia = 7 SchenegenCountries = 8) Mandatory
5 CoverageDetails[] Array Mandatory
5.1 Coverage String Define the Insurance cover area ( Ex: 24/7 Worldwide Medical Assistance, Travel Assistance, ravel Insurance, Dental Cover Mandatory
5.2 SumInsured Decimal Define the service wise cover amount (Ex Dental Cover -$250, Personal Accident-$10000) Mandatory
5.3 Excess Mandatory
6 PlanCategory Define the category of plan whether its Domestic plan. Possible values can be DomesticTravelPolicy = 1, OverseasTravelInsurance = 2 Mandatory
7 PremiumList[] Array Mandatory
7.1 PassengerCount Integer Define no of passenger in policy Mandatory
7.2 MinAge Integer Define minimum age of passenger for which policy is valid Mandatory
7.3 MaxAge Integer Define maximum age of passenger passenger for which policy is valid Mandatory
8 Price Array Price detail of the Insurance Mandatory
8.1 Currency String Currency in which minimum Insurance price is returned Mandatory
8.2 GrossFare Decimal Gross fare of the policy Mandatory
8.3 PublishedPrice Decimal (Publish price is the price that has to charge from customer it inclusive of all the taxes ) Mandatory
8.4 PublishedPriceRoundoff Decimal Publish Price (rounded off) of Insurance policy (This price is for display purpose only) Mandatory
8.5 OfferedPrice Decimal OfferedPrice of Insurance policy Mandatory
8.6 OfferedPriceRoundOff Decimal Offered Price (rounded off) of Insurance policy (This price is for display purpose only) Mandatory
8.7 CommissionEarned Decimal Commission applicable onpolicy that is the actual earning of agency on the policy Mandatory
8.8 TdsOnCommission Decimal TDS applicable on Commission Earned on policy Mandatory
8.9 ServiceTax Decimal Service tax applicable on the Policy Mandatory
8.10 SwachhBharatTax Decimal SwachhBharatTax applicable on the Policy Mandatory
8.11 KrishiKalyanTax Decimal KrishiKalyanTax aplicable on the Policy Mandatory
8.12 CancellationCharge Decimal CancellationCharges applicable on the Policy Mandatory
9 PolicyStartDate Date Time Insurance start date Format: (dd/mm/yyyy) Mandatory
10 PolicyEndDate Date Time Insurance End date Format: (dd/mm/yyyy) Mandatory
