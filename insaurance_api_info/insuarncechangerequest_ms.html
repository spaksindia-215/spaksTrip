Insurance SendChangeRequest

This method is used to cancel a confirm policy.

Service URL (REST) to be used: https://InsuranceBE.tektravels.com/InsuranceService.svc/rest/SendChangeRequest

Insurance SendChangeRequest Request
Level Parameter Data Type Description Mandatory/ Optional
1 EndUserIp String IP Address of the end user Mandatory
2 TokenId String Token ID Mandatory
(Token Id to be echoed back from Authenticate response)
3 BookingId Integer Unique booking id received in book response of the successful booking Mandatory
4 RequestType Enumeration Request Type Mandatory Possible values can be - 3 for Refund
5 Remarks String Remarks for cancelling the booking Mandatory
Remarks for cancelling the booking
Insurance SendChangeRequest Response
Level Parameter Data Type Description Mandatory/ Optional
1 TraceId String Trace ID Mandatory
2 ResponseStatus Enumeration Response status Mandatory NotSet = 0, Successfull = 1, Failed = 2, InValidRequest = 3, InValidSession = 4, InValidCredentials = 5
3 Error
3.1 ErrorCode Integer Code corresponding to the Error Mandatory
3.2 ErrorMessage String Contains error message Mandatory (Will contain error description corresponding to ErrorCode. Error Message would be blank in case ErrorCode=0)
4 PassengerChangeRequest[] Array Mandatory
5 CancellationCharge Decimal Cancellation charges applicable on booking Mandatory
6 ChangeRequestId Integer ChangeRequestId Mandatory
7 ChangeRequestStatus Enumeration ChangeRequestStatus Mandatory
8 CreditNoteCreatedOn Date Time Credit note creation date Format: (dd/mm/yyyy) Mandatory
9 CreditNoteNo String Credit note number Mandatory
10 KrishiKalyanCess Decimal KrishiKalyanCess Mandatory
11 PolicyNo Integer Policy number of policy which user want to cancel Mandatory
12 Passenger Name String Name of passenger whose policy is getting cancel Mandatory
13 RefundedAmount Decimal Refund amount after deducting cancellation charges Mandatory
14 Remarks String Remark of cacellation Mandatory
15 SwachhBharatCess Decimal SwachhBharatCess Mandatory
